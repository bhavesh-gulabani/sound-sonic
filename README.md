# SoundSonic

A modern ecommerce application which sells consumer electronics, primarily focusing on audio products.

## Technology stack

- NextJS
- Sanity CMS

## Functionalities

- Browse products
- Add/remove products to/from cart
- Order products
- Pay via Stripe
